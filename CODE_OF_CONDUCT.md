# 行为准则

本项目要求参与者保持尊重、克制和建设性的协作方式。

参与者应当：

- 聚焦技术事实、用户影响和可验证证据
- 可以质疑方案，但不要攻击个人动机或身份
- 避免骚扰、威胁、人身攻击、歧视性语言和恶意引战
- 不要在公开渠道发布私有信息、凭据、成员数据或未公开安全细节
- 尊重维护者对范围、优先级、发布节奏和安全响应的管理

维护者可以隐藏或删除让项目协作变得不安全、不可持续或明显偏离主题的评论、issue、PR 或贡献者权限。

如需报告行为准则问题，请使用仓库所有者列出的私有维护者渠道。涉及安全漏洞或敏感数据时，请同时遵循 [SECURITY.md](SECURITY.md)。
